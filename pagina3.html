Principais Comandos e suas Finalidades com o uso do Flow

Inicialização


# git flow init - Começa o uso do git-flow fazendo sua inicialização dentro de um repositório git existente.

Funcionalidades/features


# git flow feature start MYFEATURE - Começa uma nova funcionalidade. Esse comando cria um novo branch da funcionalidade baseado no 'develop' e alterna para ele.
# git flow feature finish MYFEATURE - Finaliza o desenvolvimento de uma funcionalidade. Esse comando mescla MYFEATURE no 'develop', remove o branch da funcionalidade e volta para o branch 'develop'.
# git flow feature publish MYFEATURE - Publica uma funcionalidade para o servidor remoto, assim ela pode ser utilizada por outros usuários.
# git flow feature pull MYFEATURE - Obtêm uma funcionalidade publicada por outro usuário e acompanha as alterações remotas.

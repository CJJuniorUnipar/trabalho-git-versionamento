Comandos e suas finalidades sem o uso do Flow

$ git init # inicializa um repositório local
$ git branch "nome Branch" # cria uma nova linha de desenvolvimento (branch)
$ git checkout "nome Branch" # navegação entre branches
$ git add "FILE" # adiciona um arquivo à área de preparação (stage)
$ git rm "FILE" # remove um arquivo da stage
$ git status # lista as alterações realizadas
$ git commit # consolida localmente as alterações adicionadas à stage
$ git pull "REPOSITORY" "BRANCH" # atualiza a cópia local com as alterações remotas
$ git push # públicas as alterações locais para a remota
$ git clone # copia um repositório remoto
